Mui 投票
========

一款基于云原生开发的投票系统，依托于 Vercel 的部署环境。

使用的技术和产品：

1. Nuxt.js
    1. 提供 Vue 开发环境
    2. 提供 SSR 能力
    3. 提供 API 环境
2. Vercel
    1. 提供部署环境和各项网站能力
    2. 提供 Edge Function
    3. 免费额度足够用很久
3. Upstash
    1. Redis
    2. 免费额度足够用很久
4. TiDB Cloud Serverless
    1. HTTP endpoint
    2. 免费额度足够用很久 
5. 其它
    1. TailwindCSS
    2. DaisyUI
    3. GitHub
    4. pnpm
